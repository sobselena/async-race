## Getting Started
Follow these instructions to run the project locally.

### Server Installation

1. **Clone the repository**
```bash
git clone https://github.com/mikhama/async-race-api.git
```
2. Go to the project folder
  ```bash
cd async-race-api
```
3. **Install dependencies**
```bash
npm install
```

4. **Start the server**
```bash
npm run start
```

### App Installation

1. **Clone the repository**
```bash
git clone https://github.com/sobselena/async-race.git
```

2. **Install dependencies**
```bash
npm install
```

3. **Start the app**
```bash
npm run dev
```
